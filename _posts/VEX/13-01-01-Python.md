## Functions

`def` - define function  
`return` - function's return value. How function send result to fn. caller

```python
def foo(name, age):
    #(code inside fn must be indentet)
    return name + ", you are " + age
```
Call function with arguments:
```python
foo("Adam", "30")
# Hello Adam, you are 30

print( "Hello, " + foo("Anna", "25") + " Have a nice day." )
# Hello, Anna, you are 25 Have a nice day.
```

Compare:
```python
def max_num(num1, num2, num3):
    if num1 >= num2 and num1 >= num3:
        return num1
    elif num2 >= num1 and num2 >= num3:
        return num2
    else:
        return num3

print(max_num(1,6,9))
# 9
```
