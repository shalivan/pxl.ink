## Functions

`def` - define function  
`return` - function's return value. How function send result to fn. caller

```python

    #(code inside fn must be indentet)
    return name + ", you are " + age
```
Call function with arguments:
```python

```

# Files
`open()` - `a` - append / `w` - write / `r` - read / `r+` - read write   
`close()` - close every time you open

##### Read

```python
file = open('file.txt', 'r')
print(file.read())
file.close() # close everytime you open
```
```python
file = open('file.txt', 'r')
print('Is file readable: ' + str(file.readable())) # is readable
print('Opened file info: ' + str(file))
print('Read lines in file: ' + str(file.readline())) # print first line of file
print('Read lines in file: ' + str(file.readline())) # this will print second line
```

```python
file = open('file.txt', 'r')
for file in file.readlines():
    print(file)
```
